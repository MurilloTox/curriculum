# Curriculum Vitae - Javier Murillo
## QA Automation Engineer | Mobile Specialist

### Objetivo del Proyecto
El objetivo principal de este proyecto es el diseño, desarrollo e implementación de un framework de automatización **End-to-End (E2E)** robusto y escalable para las aplicaciones móviles de **BanReservas** (Android e iOS). 

A través del uso de tecnologías modernas, este proyecto busca:
* **Garantizar la calidad:** Validar flujos críticos de negocio de forma automatizada para reducir el riesgo de regresiones.
* **Optimización Multiplataforma:** Mantener una base de código única y eficiente que maneje las particularidades de **Android (UIAutomator2)** e **iOS (XCUITest)** mediante estrategias de locators avanzadas (XPath relativos y iOS Predicate Strings).
* **Mantenibilidad y Tipado:** Utilizar **TypeScript** con un enfoque de tipado estricto para eliminar errores en tiempo de ejecución y facilitar la escalabilidad del framework.
* **Comunicación Efectiva:** Implementar **Cucumber (BDD)** para que los casos de prueba sirvan como documentación viva y clara tanto para perfiles técnicos como de negocio.
